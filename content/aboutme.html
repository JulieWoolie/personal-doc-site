What's with this website design?

I just wanted a website where I could write stuff in HTML and have it show up looking like an official A4 document. So I made this website, with simple, kinda ugly styling because I like it.

Copy content as LaTeX?

Yeah, when I first made this website, I hosted it on my PC for just me, so I needed to turn the stuff I wrote into PDFs in order to send them to others, so I wrote a HTML to LaTeX converter
